# Potato Weather!

**Potato Weather** is a Cross-platform weather app for Android, Windows, Linux, and possibly more!

Written in Flutter

# Features:
**Radar**: Radar supplied by the National Weather Service and National Oceanic Atmospheric Administration.

**Emergency Alert System**: Any EAS alerts for your area will be displayed.

**NOAA Weather Radio**: Out of Service currently. Please let me know if you know of any mp3 streams for the weather radio!!!
